# TetoOS - минимальная операционная система

TetoOS - это учебное ядро, написанное на языке Си и ассемблере. Проект создан для изучения низкоуровневого программирования, работы с видеопамятью, клавиатурой и командной строкой.

## Возможности

- Загрузка через Multiboot (GRUB / qemu)
- Текстовый терминал с курсором
- Обработка клавиатуры (английская раскладка)
- Командная строка:
    - `help` - список команд
    - `clear` - очистка экрана
    - `reboot` - перезагрузка системы
    - `poweroff` - выключение 
- Поддержка Backspace и Enter
- Автоматическая прокрутка экрана

## Сборка и запуск

### Требования

- `nasm`
- `gcc` (с поддержкой `-m32`)
- `ld`
- `make`
- `qemu` (для запуска)
- `vbox` для запуска в виртуальной машине

### Сборка для QEMU

```bash
make clean
make
```

### Запуск в QEMU

```bash
make run
```

### Создание ISO для VirtualBox (уже есть готовый образ tetos.iso)

```bash
./makeiso.sh
```

## Структура проекта

TetoOS/
